GeoRITM 2.4

Новый облачный сервис для мониторинга мобильных и стационарных объектов.

Единый, удобный интерфейс для контроля состояния объектов и обработки тревог.

Выбор типа объектов

В главном меню можно переключить режим отображения списка мобильных или стационарных объектов.

Все выбранные мобильные и стационарные объекты будут одновременно отображаться на карте.

Объекты на карте

Цвет иконки объекта сообщает о его состоянии.

Зеленый - снят с охраны
Синий - в охране
Желтый - есть неисправность
Серый - не на связи
Красный - не сброшенная тревога

Настройка приборов

Удобная и быстрая настройка новых приборов непосредственно из интерфейса системы мониторинга.

Для настройки Contakt GSM-14, Voyager 2N-ГЛОНАСС и Voyager 4N теперь не обязательно пользоваться кабелем настройки или подключаться дистанционно с помощью модема.

Мастер настройки

Мастер настройки поможет Вам подключить новый прибор.

Для Voyager 2N, 4N и Contakt-14 доступно автоматическое добавление включенного прибора в Вашу учетную запись по его IMEI-коду.

Сводная информация

Страница с настраиваемыми индикаторами для отображения состояния системы.

При клике на индикатор будут выбраны соответствующие ему объекты.

Мобильное приложение

Мобильное приложение для смартфонов на Android и мобильная версия сайта для любых мобильных устройств.

Изображение объекта

В карточке объекта добавлена возможность загрузки изображений в форматах jpg и png.

Загружайте фото объекта и схемы установки оборудования для мобильных и стационарных объектов.

Поиск ГБР

При обработке тревоги оператором охранного предприятия система сама найдет ближайшие группы быстрого реагирования и предложит кратчайшие маршруты движения.
